# GestionVoitures

Télécharger le projet

Builder le projet avec : gradlew build 

Convertir le projet en un projet Eclipse : gradlew eclipse

Importer le projet sous Eclipse : File -> import -> general -> existing project into workspace ...

Lancer le main du projet dans src/main/java : fr.descartes.GestionVoituresApplication

Accéder au projet dans un nagigateur : localhost:8080/static/index.html
